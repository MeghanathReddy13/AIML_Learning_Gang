How Hidden Neurons Learn Patterns

A simplified student-friendly walkthrough using a small house dataset.

2. The network only sees input features

Input features

X = [Size, Bedrooms, Age]
The network does not magically know neighborhood, school rating, or income level unless we provide those features.

Learned features

Hidden layers create new combinations of the provided input features.
Hidden feature = f(w₁Size + w₂Bedrooms + w₃Age + b)

3. One hidden layer with three neurons

Input Layer Hidden Layer Size ───────────────┐ h₁ Bedrooms ───────────────┼────▶ h₂ Age ───────────────┘ h₃
In a fully connected layer, each hidden neuron receives all inputs.

4. Same inputs, different weights

h₁, h₂, and h₃ are called different neurons because each has its own weight vector.

Neuron h₁

h₁ = ReLU(0.9Size + 0.8Bed − 0.7Age)

Neuron h₂

h₂ = ReLU(0.1Size + 2.5Bed + 0.2Age)

Neuron h₃

h₃ = ReLU(−0.2Size + 0.1Bed − 3Age)

5. Think of each neuron as a different perspective

h₁

Looks at size, bedrooms, and age together.

Luxury potential

h₂

Gives high importance to bedrooms.

Family-house score

h₃

Strongly reacts to newer homes.

Modernity score

These names are for teaching. The model only learns numbers, not labels.

6. Calculate hidden values for one house

House C

Size = 2200, Bedrooms = 4, Age = 5
h₁ = ReLU(1980 + 3.2 − 3.5) = 1979.7
h₂ = ReLU(220 + 10 + 1) = 231
h₃ = ReLU(−440 + 0.4 − 15) = 0
The original row becomes a new representation: [1979.7, 231, 0]

7. The next layer sees hidden features, not raw features

Original features Hidden representation [Size, Bedrooms, Age] ───▶ [h₁, h₂, h₃] [2200, 4, 5] ───▶ [1979.7, 231, 0]
The network has transformed the data into a new feature space that may be easier for prediction.

8. Why multiple neurons?

One neuron

Can learn one useful combination of the inputs.
h₁ = one view of the house

Multiple neurons

Can learn many useful combinations at the same time.
[h₁, h₂, h₃] = multiple views
Same inputs + different weights = different learned patterns.

9. How does learning actually happen?

1. Start with random weights.
2. Make a prediction using forward pass.
3. Compare prediction with actual answer.
4. Backpropagation finds which weights caused error.
5. Update weights and repeat many times.

10. Before and after training

Before training

Weights are random, so hidden neurons are mostly meaningless.
h₁ = random combination
h₂ = random combination

After training

Weights become useful because they reduce prediction error.
h₁ ≈ luxury potential
h₂ ≈ family-house signal

11. Where does nonlinearity come from?

The weighted sum is linear, but the activation function makes it nonlinear.
z = w₁x₁ + w₂x₂ + b
h = ReLU(z) = max(0, z)
z negative ──▶ 0 z positive ──▶ z

14. Layer 2 combines Layer 1 patterns

Layer 1 outputs Layer 2 neurons h₁ ───────────────┐ g₁ h₂ ───────────────┼──────▶ g₂ h₃ ───────────────┘
Layer 2 does the same thing again: it receives all previous outputs, uses different weights, applies activation, and creates more abstract features.

15. Simple mental model

Raw inputs Size, Bedrooms, Age ↓ Layer 1 learns simple combinations Large? Many bedrooms? New? ↓ Layer 2 combines those patterns Luxury candidate? Family home? ↓ Output Luxury: Yes or No

16. Key takeaway

A hidden neuron receives all inputs from the previous layer, but it has its own weights.

Different weights create different combinations.

Activation functions turn those combinations into nonlinear pattern detectors.

Multiple neurons learn multiple useful representations of the same data.
